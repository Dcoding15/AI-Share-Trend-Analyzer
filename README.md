# 📈 AI Share Trend Analyzer

A full-stack AI-powered stock market trend analysis application built to analyze share market data, visualize trends, and provide predictive insights using machine learning techniques.

The project combines financial data analysis, AI/ML models, and interactive visualizations to help users better understand stock market movements and trends.

## 🚀 Installation & Setup

### Prerequisites
- Python 3.8+
- pip
- Virtual Environment (recommended)

### Setup

```bash
# Clone the repository
git clone https://github.com/Dcoding15/AI-Share-Trend-Analyzer.git

# Navigate into project folder
cd AI-Share-Trend-Analyzer

# Create virtual environment
python -m venv venv

# Activate environment
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
